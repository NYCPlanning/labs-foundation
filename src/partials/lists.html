{{{headerify 'h3' 'Lists'}}} {{#markdown}} #### Unordered lists Use the `
    ` tag to list things if the order of the items doesn't matter. ``` html
    • List item
    • List item
    • This list item contains another list
      • Nested list item
      • Another nested list item
    • List item with a much longer description or more content. Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
    ```
    • List item
    • List item
    • This list item contains another list
      • Nested list item
      • Another nested list item
    • List item with a much longer description or more content. Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
    #### Ordered lists The `
      ` tag defines lists with an explicit order, marked with numbers by default. The optional `type="[1,A,a,I,i]"` attribute defines the type of the list item marker. ``` html
      1. Bicycles
      2. Bike racks
      3. Bike lanes
        1. Sharrows
        2. Striped
        3. Buffered
        4. Protected
      ```
      1. Bicycles
      2. Bike racks
      3. Bike lanes
        1. Sharrows
        2. Striped
        3. Buffered
        4. Protected
      #### Un-bulleted lists The `
        ` is a bulleted list and `
          ` is a numbered list by default, but you can add the class `.no-bullet` to remove the bullets or numbers. This allows lists to have semantic markup and custom styles so they do not look like lists (useful for lists such as menu items). ``` html
          • List item
          • List item
            • Nested list item
            • Nested list item
          • List item
          ```
          • List item
          • List item
            • Nested list item
            • Nested list item
          • List item
          #### Definition lists The `
          ` tag is used to display name-value pairs, like metadata or a dictionary definition. Each term (`
          `) is paired with one or more definitions (`
          `). ``` html
          Time
          The indefinite continued progress of existence and events in the past, present, and future regarded as a whole.
          Space
          A continuous area or expanse that is free, available, or unoccupied.
          The dimensions of height, depth, and width within which all things exist and move.
          ```
          Time
          The indefinite continued progress of existence and events in the past, present, and future regarded as a whole.
          Space
          A continuous area or expanse that is free, available, or unoccupied.
          The dimensions of height, depth, and width within which all things exist and move.
