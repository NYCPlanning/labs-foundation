#### Background colors Use the `.bg-[color]` classes to set an element's **background color** and its default **text color** (defaulting to black or white, depending on which is more accessible). Text color classes override the default text color set by background color classes. ```

This element has a yellow background, which defaults to black text.

This element has a dark blue background, which defaults to white text.

This element has a light chartreuse background and dark green text (which overrides its default black text).
