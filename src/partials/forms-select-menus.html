{{{headerify 'h3' 'Select menus'}}} {{#markdown}} Select menus combine choices into one condensed menu. However, when the user needs to see all available choices, radio buttons are often more user-friendly. For long lists, consider using a typeahead autocomplete component. ``` ```
#### Multi-select The `multiple` attribute allows more than one option to be selected. Use the `size` attribute to set how many options are visible at once (defaults to 4). However, since UX can vary by OS/browser and multiple selection may not be intuitive, checkboxes are often more user-friendly. ``` ```
