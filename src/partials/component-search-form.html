{{{headerify 'h3' 'Search form'}}} {{#markdown}} It's a standard UI convention for search inputs to have a magnifying glass icon. This clearly communicates the function of the input. This is the one case where it's acceptable to use placeholder text. However, you must also include a label for accessibility — the `.show-for-sr` class can be added to the label so it's hidden on screen but still available for screen readers. Use the `.search-form`, `.search-input`, and `.search-icon` classes to create a search component where the icon is positioned over the input. The `.large-padding-right` prevents the icon from overlapping the input's text. ```
