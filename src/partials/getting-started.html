{{#markdown}} There are a number of ways to incorporate the NYC Planning Digital Style Guide into your projects, depending on your technology stack and requirements. {{/markdown}}
{{{headerify 'h3' 'Installation'}}} {{#markdown}} This package is available via [NPM](TODO: add link), and includes all of the source **Sass** and **JavaScript** files (as well as compiled CSS and JavaScript). Install this package: ``` npm install nyc-planning-style-guide ``` #### Sass Edit your `app.scss` to import/include NYC Planning Style Guide files: ``` // NYC Planning variables (colors, Foundation utilities and settings) @import 'nyc-planning-style-guide/variables'; // Foundation @import 'foundation-sites/foundation'; // Include all or a subset of Foundation modules @include foundation-everything; // Import all or a subset of NYC Planning modules @import 'nyc-planning-style-guide/all-modules'; // Custom app modules and styles @import '_my-app-module'; .peanut-butter { color: $orange-muted; } ``` #### JavaScript JavaScript is optional, but many interactive components require it. You can either load this package's JS (which is a compiled combination of Foundation's JavaScript and its dependencies) or individually load jQuery, What Input, and specific Foundation JavaScript components and then initialize Foundation. For more information on setting up custom JavaScript, see the [Foundation JavaScript documentation](https://foundation.zurb.com/sites/docs/javascript.html). {{/markdown}}
{{{headerify 'h3' 'Ember and Labs UI'}}} {{#markdown}} Labs UI is an EmberJS addon that contains common components used in NYC Planning digital products. It consumes Sass files from this package to style its components. Since Ember handles behavior and state management, Labs UI does not include any JavaScript from this package. For more information, see the Labs UI documentation. {{/markdown}}
{{{headerify 'h3' 'Download'}}} {{#markdown}} The simplest way to incorporate the NYC PLanning Digital Style Guide into your project is to download the compiled assets and include them in your code. - **Style**: nyc-planning.css  - **JavaScript**: nyc-planning.js  Be sure to include the `.no-js` class on your `html` tag.
