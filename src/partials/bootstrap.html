This site references the Foundation framework throughout, but you can also use the Bootstrap framework.

I have attempted to translate the settings used in the Foundation files into a Bootstrap template. I have also attempted to port over as much of possible any parts of the guide where syntax is specific and differs from Bootstrap defaults so that either can be used. For example:
