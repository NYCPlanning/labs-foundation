{{{headerify 'h3' 'Text inputs'}}} {{#markdown}} The following input types create a text input: `text`, `date`, `datetime-local`, `email`, `month`, `number`, `password`, `search`, `tel`, `time`, `url`, and `week`. ``` ```
In some desktop browsers, certain types — `number`, `date`, `search`… — have controls in them which allow the user to increment/decrement the text inside the field, select it from a dropdown, or clear it. When adding custom controls, be sure they don't conflict with these browser-specific controls. Adding the `.no-controls` class will remove most of them. {{/markdown}}
