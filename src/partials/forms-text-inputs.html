{{{headerify 'h3' 'Text inputs'}}} {{#markdown}} The following input types create a text input: `text`, `date`, `datetime-local`, `email`, `month`, `number`, `password`, `search`, `tel`, `time`, `url`, and `week`. ``` ```
In some desktop browsers, certain types — `number`, `date`, `search`… — have controls in them which allow the user to increment/decrement the text inside the field, select it from a dropdown, or clear it. When adding custom controls, be sure they don't conflict with these browser-specific controls. For example, if you add a custom clear button to your search, disable the browser clear buttons: ```CSS .custom-search-input::-ms-clear, .custom-search-input::-webkit-search-cancel-button { display: none; } ``` {{/markdown}}
